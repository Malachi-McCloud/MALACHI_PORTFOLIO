Power Platform Workflow Hub

Inspection Report Analytics Pipeline

Problem: Time-based reports were completed without generating insights. Audits required archived forms, but no centralized analytics existed.

Solution: Built a Power App front-end mimicking the form, backed by a SharePoint List. Normalized data was archived and used to refresh a Power BI dataset, enabling training readiness analytics.

Tools: Power Apps, SharePoint Lists, Power Automate, Power BI

Impact: Enabled audit-ready archiving and live analytics for compliance and readiness tracking.

Multi-Team Approval & Signature Workflow

Problem: Requests requiring multi-team approval were lost due to manual handling and licensing barriers for digital signatures.

Solution: Created a Power App for structured case submission. Triggered flows created SharePoint Document Library entries and sent Microsoft Approvals with synced OneDrive signing instructions. Naming conventions and error handling ensured traceability.

Tools: Power Apps, SharePoint Document Libraries, Power Automate, Microsoft Approvals

Impact: Reduced lost submissions and streamlined multi-team approvals with secure document handling.

Cloud-to-Desktop Voucher Extraction Pipeline

Problem: Manual voucher processing led to repetitive work and high error rates.

Solution: Power App submission triggered a cloud flow that queued a Desktop flow on a Power Platform VM. The Desktop flow extracted text from PDFs and updated a SQL table. Staff were notified to verify extraction status.

Tools: Power Apps, SharePoint, Power Automate Cloud & Desktop, Power Platform VM, SQL

Impact: Automated voucher processing and reduced human error with secure, scalable architecture.

Robotic Process Automation

Loan Data Consolidation & Optimization

Problem: Loan data was split across three CSV/XLSX files with poor readability and a fragile legacy process that failed 50% of the time and consumed excessive VM resources.

Solution: Collaborated with RPA Lead to rebuild the process using an ODBC Desktop SQL driver. Structured the data into a datatable (2D array in Automation Anywhere) and exported to a preformatted Excel file. Also prototyped an Excel-only model using formulas and RPA, though it was resource-intensive.

Tools: Automation Anywhere, SQL ODBC Driver, Excel

Impact: Reduced runtime, improved reliability, and enabled structured analysis of loan performance.

Taskbot Status & Error Handling Framework

Problem: RPA flows failed unpredictably, and email-based error handling broke under network/API disruptions, leaving failures undetected.

Solution: Built a reusable taskbot that logged task status (Started, Finished, Waiting) to a SQL table. Added pre-launch status updates and fallback notifications for missed transitions. Enabled a control panel view of task health across deployments.

Tools: Automation Anywhere, SQL, Cloud Automation Framework

Impact: Improved visibility, reduced silent failures, and created a scalable error-handling backbone.

Rate Scraper & Distribution Bot

Problem: Teams needed accurate financial rates from WSJ and other sites, but manual retrieval was slow and error-prone.

Solution: Designed an RPA taskbot using Selenium to scrape rate tables, process by date, and export to Excel. Automated email distribution to relevant teams. Planned SQL integration for historical tracking and dashboarding.

Tools: Automation Anywhere, Selenium, Excel, Email Automation

Impact: Delivered timely rate updates and laid groundwork for scalable financial data ingestion.
