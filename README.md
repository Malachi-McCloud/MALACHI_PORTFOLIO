# Resume Portfolio

Mission-focused showcase of reproducible analytics, secure deployment, and effective storytelling for federal roles.

## Overview

This portfolio is built to demonstrate full-stack readiness and a wide skillset built for Cleared Federal Roles where flexibility and adaptability is key.

- Modular resume layout with tactical styling for message delivery
- Analytics dashboard scaffolded for Seaborn/Pandas visualizations
- Secure deployment pipeline via Azure Static Web Apps and GitHub Actions
- Placeholder tiles for reproducible, mission-aligned projects

## Tech Stack

**Frontend**
- HTML5, CSS3
- Responsive Grid Layout

**Backend**
- Flask (Blueprints ready)
- Python (Seaborn, Pandas - in progress)

**Deployment**
- Azure Static Web Apps
- GitHub Action (CI/CD)

**Design Philosophy**
- Modular structure
- Reproducible builds
- Missions clarity
